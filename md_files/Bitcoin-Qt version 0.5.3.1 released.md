# Bitcoin-Qt version 0.5.3.1 released

16 March 2012

Bitcoin-Qt version 0.5.3.1 for Windows is now available for download at:
<http://sourceforge.net/projects/bitcoin/files/Bitcoin/bitcoin-0.5.3/>

This is a bugfix-only release based on 0.5.1.

Please report bugs using the issue tracker at GitHub:
<https://github.com/bitcoin/bitcoin/issues>

## BUG FIXES

  * Fixed a potentially critical security vulnerability in Windows versions of Bitcoin-Qt.
